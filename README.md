# Tarea_1
Estimar los efectos de la infraestructura pública sobre la desigualda a nivel regional en el Perú para el período 2004–2023

El objetivo de este análisis es estimar los efectos de la infraestructura pública sobre la desigualda a nivel regional en el Perú para el período 2004–2023 (empleando para medir la desigualdad, el Indice de Gini -construido para cada región del país-). El análisis se efectuará empleando la información de la ENAHO para el período mencionado, con la finalidad de disponer de indicadores físicos sobre la infraestructura de transporte, electricidad y telecomunicaciones. Las demás variables de control a ser utilizadas se obtienen de los Compendios Estadísticos del INEI correspondiente a los años antes mencionado
